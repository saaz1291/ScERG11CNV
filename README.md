# ScERG11CNV
Codes for figures and analysis, and some input data, for my research on the experimental evolution of ERG11 copy number variants in S cerevisiae under fluconazole selection

All_outputs_analysis_2026Jan9.Rmd: Fig. 1C - Analysis of growth data of control/anccestral strains. Input = Control_growthcurve_output_2026Jan9.xlsx

Allruns_timecourse_presentationgraphs_6June2025.Rmd: Fig. 2 and Fig. S2 - Plotting CNV evolution from gating data. Input = Apr2025_allruns.xlsx, dependency = plotprocessing_passage.R

All_outputs_analysis_2026Mar27.Rmd: Fig. 3 - analysis of growth rates of CNV vs control strains. Input = All_growthcurve_output_2025Apr7.xlsx

Depth_visualisation_comparison_May82025.Rmd: Fig. 4 - visualisation of genome-wide read depth, comparing the ancestral strain with one CNV strain

perbase_groupvisualisation-allcond_selected.Rmd: Fig. S3 - visualisation of genome-wide read depth, comparing multiple strains

Doseresponse_all_Aug6.Rmd: Fig. S1 - getting IC50 from control strain growth data. Input = All_growthcurve_output-Aug6.xlsx

growth_curves_Apr29.Rmd: Analysis of raw growth data, using a single batch as an example. Inputs = Apr29_meta.csv and Apr29_growth.csv
